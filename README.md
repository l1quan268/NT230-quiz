# NT230 - Malware Mechanisms Quiz Web App

An interactive, user-friendly web application designed for exam preparation for the **NT230 - Malware Mechanisms** course at UIT. 

🚀 **Live Demo:** [Click here to practice](https://l1quan268.github.io/NT230-quiz/) 

---

## 📌 Features
- **Comprehensive Question Bank:** Features 310 multiple-choice questions covering extensive course knowledge from lectures L02 to L07.
- **Key Modules Covered:** Executable Format, Virus, Worm, Botnet/APT, Stealth Malware, and Anti-Dynamic Analysis.
- **Dynamic Learning Experience:** Generates randomized questions per session to maximize retention, accompanied by detailed explanatory popups for every answer selection.
- **Modern UI/UX:** Built with Tailwind CSS, supporting dark/light mode toggle, dynamic progress tracking, and score summary.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Tailwind CSS via CDN), JavaScript (ES6)
- **Deployment:** GitHub Pages
